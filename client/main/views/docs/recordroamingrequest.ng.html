Overview

POST /roaming
Request a ride

Request

Parameter Type Description
id String Unique identifier for your request
type String Values: [sedan, suv]

Response

Parameter Type Description
result string Result message
result_code int Result code

$curl -X POST -d'{"test": "x"}' https://sandbox.tripthru.com/roaming?access_token=81b61ef32f308680af46fac749b1cda2f9cadca09c1e866c9fe841eda555c4dc

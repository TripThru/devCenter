Overview

The chosen authentication method is OAuth2's Client Credentials Grant flow.

After registration you will receive your credentials client_id and client_secret. You'll use them to request an access token to authorize subsequent API calls.

POST /oauth/token
Use your client_id and client_secret to request an access token

Request

Parameter Type Description
grant_type String You must send client_credentials as value
client_id String Your client_id
client_secret String Your client_secret

Response

Parameter Type Description
access_token string Your new access token
expires_in int Duration of token validity

Example code

$curl -d 'grant_type=client_credentials' -d 'client_id=x' -d 'client_secret=x' -X POST https://sandbox.tripthru.com/oauth/token
 

var request = require("request");

request({
  uri: "https://sandbox.tripthru.com/oauth/token",
  method: "POST",
  timeout: 10000,
  followRedirect: true,
  maxRedirects: 10,
  form: {
    grant_type: "client_credentials",
    client_id: "x",
    client_secret: "x"
  }
}, function(error, response, body) {
  console.log(body);
});
